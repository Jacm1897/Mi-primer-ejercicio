Infografía Interactiva: Top 5 Modelo PjBBL-DT
Una infografía web interactiva y visualmente atractiva, construida exclusivamente con HTML y CSS puro (sin JavaScript), que resume los 5 hallazgos más importantes del estudio académico sobre la integración del Aprendizaje Mixto Basado en Proyectos y el Pensamiento de Diseño (PjBBL-DT).

✨ Características Principales
Diseñada bajo el paradigma visual "Modelo RIOSEÑO", la infografía incluye:

Estructura Semántica: Lista ordenada (<ol>) con 5 elementos, cada uno compuesto por una subestructura (Título <h3> + Descripción <p>).
Efecto Principal (Hover): Cada ítem posee un fondo semicircular (border-radius: 100px 100px 0 0). Al pasar el cursor, este fondo rota exactamente 180deg transformándose en un círculo/óvalo completo.
Efecto Secundario: Un fondo inferior compuesto por SVG que simula olas con una animación horizontal infinita y fluida (@keyframes waveMotion).
Ritmo Visual: Tipografía estricta de 18px, fuente sans-serif, espaciado generoso (gap: 35px) y una paleta de colores vibrantes y neón sobre fondo oscuro.
Cero Dependencias: No utiliza librerías externas, frameworks ni JavaScript.
🛠️ Tecnologías Utilizadas
HTML5: Para la estructura semántica y la inserción de los gráficos vectoriales SVG.
CSS3:
Flexbox para el layout.
Pseudoelementos (::before) para las formas geométricas.
Variables CSS (--item-color) para la gestión de colores.
@keyframes para la animación del fondo.
🚀 Cómo ejecutarlo
Dado que es un archivo estático, no requiere servidores ni instalaciones complejas:

Descarga o clona el repositorio.
Haz doble clic en el archivo index.html (o el nombre que le hayas dado al archivo).
Se abrirá en tu navegador web predeterminado. ¡Pasa el cursor sobre los elementos para ver la magia!
